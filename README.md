# Pokédex

![FV-k_w](https://i.makeagif.com/media/7-16-2022/FV-k_w.gif)

> Pokédex desenvolvida com HTML, CSS e JavaScript, consumindo a [PókeAPI](https://pokeapi.co/). A tela exibe os nomes dos Pokémons e, ao clicar em um, são apresentadas imagens do Pokémon escolhido.
